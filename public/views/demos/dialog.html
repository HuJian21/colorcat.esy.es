dialog使用说明

使用前需要引用dialog.css和dialog.js文件，依赖jquery

参数 值 默认设置 描述 是否可选
eventType click/callback click 触发类型: click:需要ClickButton对象触发 / callback:在一个函数完成时调用 可选
ClickButton 对象 -- 触发对象 eventType为click时必须填写
Title true/false false 是否有标题: true:有标题 / false:无标题 可选
titleType text/html text 标题内容类型: text / html标签 可选
titleContent 内容 -- title的内容 有标题时必须填写
contentType text/html text content内容类型: text / html标签 可选
content 内容 -- content的内容 必须填写内容
buttonNum 数字 1 数字代表button的个数 可选
buttonText 对象 buttonNum=1:确定,buttonNum=2:确定，取消 text代表按钮的文字,text1为第一个按钮，text2为第二个按钮,依次... 可选（3个以上按钮必须填写文字）
clickFn 函数 -- 弹窗里第一个按钮点击时触发的函数 可选

demo1: 默认的弹窗

alertWindow({ ClickButton: $('.js-button'), content: '您的意见或建议已经提交，<br/>感谢您对我们的支持！' });

demo2: 两个按钮的弹窗且自定义按钮文字

alertWindow({ ClickButton: $('.js-button'), content: '您的意见或建议已经提交，<br/>感谢您对我们的支持！', buttonNum: 2, // button的个数 buttonText: { text1: '确定', text2: 'Cancel', } });

demo3: 带title且titleType为html的弹窗

alertWindow({ ClickButton: $('.js-button'), Title: true, titleType: 'html', titleContent: '<p class="windowtitle">您将支付<span class="integral">5000积分</span>兑换商品!</p>', content: '您的意见或建议已经提交，<br/>感谢您对我们的支持！', buttonNum: 2, // button的个数 buttonText: { text1: '确定', text2: 'Cancel', } });
